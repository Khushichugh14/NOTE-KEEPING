The Note Keeping Website** is a lightweight and intuitive web application that allows users to create, update, delete, and organize notes efficiently. It offers a clutter-free interface for storing personal thoughts, to-do items, and reminders—accessible anytime, anywhere.
 Tech Stack:
Frontend: Angular, HTML5, CSS3, Bootstrap

Backend: Spring Boot (Java), RESTful APIs 

Database: MySQL 

Authentication: JWT-based login system (optional)
User Registration & Login (Optional)

Secure authentication system

Access notes only after logging in

Create Notes

Add notes with a title and description

Notes are saved in the database for persistence

Edit & Delete Notes

Modify existing notes

Delete notes permanently

Search Notes

Search functionality to filter notes by keywords
